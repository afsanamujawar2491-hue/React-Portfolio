# My Portfolio Website

## Overview
A personal portfolio website built with React to showcase projects, skills, and contact information.

## Features
- Responsive design
- Sections: About Me, Projects, Skills, Resume, Contact
- Lazy loading for components and images
- Optimized for fast performance

## Deployment
- Built with `npm run build`
- Deployed GitHub Pages
- Custom domain and SSL configured

## Challenges
- Ensuring mobile responsiveness
- Optimizing large images
- Lazy loading React components

## Future Improvements
- Integrate backend for contact form
- Add animations and transitions
